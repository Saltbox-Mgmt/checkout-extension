📋 How to Get Session ID:
  1. Open your Salesforce org in a new tab
  2. Press F12 to open Developer Tools
  3. Go to Application tab → Cookies
  4. Find your Salesforce domain
  5. Copy the 'sid' cookie value
  6. Paste it in the Session ID field above
Troubleshooting:
  • Make sure you're logged into Salesforce
  • For production: Use My Domain URL (not lightning.force.com)
  • For sandbox: Include --sandboxname in the URL
  • Session ID should be 100+ characters long
